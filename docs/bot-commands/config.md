## !a warnsbeforeban &lt;NUMBER \(Max of 50.\)&gt;

Sets the amount of warns before ban.

{% tabs %}
{% tab title="Usage" %}
* !a warnsbeforeban &lt;NUMBER \(Max of 50.\)&gt;
{% endtab %}

{% tab title="Aliases" %}
* wbb
* warns\_before\_ban
* warns-before-ban 
{% endtab %}

{% tab title="Permissions" %}
* manage server
{% endtab %}

{% tab title="Channel Requirements" %}
* none
{% endtab %}

{% tab title="Cooldown" %}
* 1 every 2 seconds per user.
{% endtab %}
{% endtabs %}
