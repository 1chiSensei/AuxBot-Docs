Welcome to AuxBot-Python's documentation!

* **AuxBot**
.. image:: images/AuxBotLogo.png
  :width: 400
  :alt: AuxBot Logo

AuxBot is multi use discord bot which has many different functions such as logging, moderation, music, memes, fun, and so many more commands. AuxBot has an advanced logging system which integrates with it's advanced moderation commands as well. Check it out by inviting the bot to your server!
