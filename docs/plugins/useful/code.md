## !a run

Runs code in a chosen language.

{% tabs %}
{% tab title="Usage" %}
* !a run &lt;LANGUAGE&gt; &lt;CODE&gt;
{% endtab %}

{% tab title="Aliases" %}
* code
{% endtab %}

{% tab title="Permissions" %}
* none
{% endtab %}

{% tab title="Channel Requirements" %}
* none
{% endtab %}

{% tab title="Cooldown" %}
* 1 every 2 seconds per user.
{% endtab %}
{% endtabs %}
