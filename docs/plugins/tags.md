## !a servertags

Returns all of the servers tags.

{% tabs %}
{% tab title="Usage" %}
* !a servertags
{% endtab %}

{% tab title="Aliases" %}
* sts
* server\_tags
* server-tags
{% endtab %}

{% tab title="Permissions" %}
* none
{% endtab %}

{% tab title="Channel Requirements" %}
* none
{% endtab %}

{% tab title="Cooldown" %}
* 1 every 2 seconds per user.
{% endtab %}
{% endtabs %}
