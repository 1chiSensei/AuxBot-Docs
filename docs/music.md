# Music

> Key:  • \[Optional arg.\] • \(Options for the arg.\) • &lt;This / Or this&gt;

### !a connect \[CHANNEL ID\]

Makes AuxBot connect to a voice channel.

{% tabs %}
{% tab title="Usage" %}

{% endtab %}

{% tab title="Aliases" %}

{% endtab %}

{% tab title="Permissions" %}
* none
{% endtab %}

{% tab title="Channel Requirements" %}

{% endtab %}

{% endtab %}
